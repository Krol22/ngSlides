Ng-slides
Angular 1.6 module for presentations
Supported animations
  • fade
  • slideLeft
  • slideRight
  • slideDown
  • slideUp
  • cascade
  • custom slides
Custom animations (1.)
Create animation object 
                            animationName : {
                                animationEnter : {
                                    transitions: [
                                      { css: 'opacity 0.9s ease-in-out', from: 0, to: 1 },
                                    ]
                                },
                                animationLeave : {
                                    transitions: [
                                      { css: 'opacity 0.9s ease-in-out', from: 1, to: 0 },
                                    ]
                                }
                            }
Custom animations (2.)
Then set this animation as current animation 
let currentAnimation = 'animationName';
